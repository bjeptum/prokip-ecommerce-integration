Connect Shopify Store

How to Connect Your Shopify Store

  1. Enter your Shopify store URL (e.g., mystore.myshopify.com)
  2. Click "Connect Store" to start the OAuth process
  3. You'll be redirected to Shopify to approve the connection
  4. Once approved, you'll be redirected back and the store will be connected
Enter your Shopify store URL without https://

Connect WooCommerce Store

How to Connect Your WooCommerce Store

  1. Go to your WordPress admin panel
  2. Navigate to WooCommerce → Settings → Advanced → REST API
  3. Click "Add Key" to create new API credentials
  4. Set permissions to "Read/Write"
  5. Copy the Consumer Key and Consumer Secret below
